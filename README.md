# Placement Java Project

A Java project for placement preparation and coding practice.

## Project Structure

```
placement-java/
├── src/
│   ├── main/
│   │   ├── java/          # Java source files
│   │   └── resources/     # Application resources
│   └── test/
│       ├── java/          # Test source files
│       └── resources/     # Test resources
├── README.md
└── .gitignore
```

## Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher
- A Java IDE or text editor

### Compilation
```bash
javac -d out src/main/java/*.java
```

### Running
```bash
java -cp out placement1
```

## Contributing
Add your placement preparation code in the `src/main/java` directory.
